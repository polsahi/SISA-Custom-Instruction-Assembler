# SISA Custom Instruction Assembler

A simple program to assemble SISA assembly with custom instructions using
the sisa-tools provided by the teachers of PEC.

## Requierements

- SISA-tools on PATH
- Python >= 3.10

## Usage

```
$ chmod +x limpiar.pl
$ chmod +x generar_test.sh
$ ./generar_test.sh test.S
```
